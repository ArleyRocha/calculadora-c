# 🧮 Calculadora em C

Projeto desenvolvido como primeiro passo na minha jornada em programação 🚀

Projeto simples de uma calculadora feita em linguagem C com interação via terminal.

## 🔧 Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Tratamento de divisão por zero
- Menu interativo com repetição (while)

## 💻 Como executar

1. Compile o código:
   
gcc calculadora.c -o calculadora
./calculadora

## 💻 Tecnologias
- Linguagem C

## 🎯 Objetivo
Praticar lógica de programação e fundamentos da linguagem C.
